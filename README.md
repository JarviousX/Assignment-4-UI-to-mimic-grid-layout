# App Shortcuts Grid

A React Native mobile application featuring a cyber night city theme with animated neon borders.

## Features

- Grid and Rows layout modes for app shortcuts
- Animated rotating gradient borders (neon blue to purple)
- Dark cyberpunk theme with neon accents
- Smooth color transitions and glowing effects

## Running the App

```bash
# Install dependencies
npm install

# Start Metro bundler
npm start

# Run on Android
npm run android

# Run on iOS
npm run ios
```

## Project Structure

- `App.tsx` - Main app component with navigation
- `src/screens/` - Screen components (HomeScreen, DetailScreen)
- `src/components/` - Reusable components (GridLayout, RowsLayout, RotatingGradientBorder)
- `src/data/` - App data and configuration
